# Changelog

## 0.1.0 — 2026-06

Initial open-source release of Personal Finances web (paired with personal-finances-express-server).

- Dashboard for transactions, categories, recurring purchases, loans, statement imports, and AI prompts
- Redux app state with manual thunks; feature UI in `src/packages/`
- Express API clients in `src/api/` (no Next.js BFF layer)
- OSS docs: README, SECURITY, wire contract, CI
